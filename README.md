# url-shortner-Go


## How to run ? 
```
// run the main file 
$ go run main.go 

// Open :8080 on your browser or curl into it 

// To Shorten a url 
$ curl "http://localhost:8080/shorten?url=https://example.com"

eg : http://localhost:8080/NfohgA 

// To Vist the Short link (redirect) 
$ curl -L "http://localhost:8080/Nf0hgA"
